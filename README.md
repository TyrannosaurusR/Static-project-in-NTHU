# Static-project-in-NTHU
此份 repo 將紀錄我在清大做過的各種數據分析專案

## 如何在特定條件透過調整習慣
> 喜好程度(1~5):1

此部分為我修習統研所的統計學習後所作出的專案作品，此份專案作品使用了 Kaggle 上記錄了學生的每天時間分配以及其學習表現的資料，較為可惜的是這是一份虛擬的資料。這份作業合作起來沒有那麼開心，組員一開始沒有很積極。當時一開始，我是想要使用真實的資料的，於是我翻過很多資料集以及論壇，可惜最後時間真的來不及，我才也接受了使用這份虛擬資料。

### 資料處理
由於這是一份虛擬的資料，所有的變數都只呈現很完美的常態分佈，且相關性都只與學習表現有關。

### 研究過程與結果
我們利用嶺回歸來探討學習的邊境遞減效應，以及利用多項式回歸得出最好的模型後，再利用線性規劃得出理論上的時間分配最佳值

## 透過機器學習模型分析紅酒品質
> 喜好程度(1~5):3

此部分為我修習清大通識中心所開立的統計思維與分析所作出的專案作品，此份專案作品一樣使用了 Kaggle 上的數據集。原本這次的專題也是想要改進先前高中所作的珊瑚礁研究，但當時一樣無法很好的結果，因此本次專案是從其中一位組員的另一份作業中改過來的。

### 資料處理
將各特徵按照品質來分類畫成堆疊圖後意外發現，其非常接近常態分布的格式，這讓資料清洗變得異常輕鬆。只需要針對每個的特徵的每個品質處理極端值即可。

### 模型訓練
除了使用相關係數外，還使用了StepAIC加上穩健選擇法，不斷重複得到最好的模型與特徵，可惜的是最後模型的R-square 只有 0.4，較為不足。

### 最佳化應用
利用Polynomial Regression的模型在使用SFS序列特徵選擇法，嘗試去最小化對人體有害的成分後得到最健康品質也最好的紅酒品質。

## 班上不同性別對音樂類型的偏好選擇
> 喜好程度(1~5):5

此部分為我修習清大統研所所開設的統計資料分析所做出來的專題作品，資料來源於班上統計。此份研究讓我感覺到從前作研究的快感，因此雖然這個作業做到半夜，但還是對於真的有在探討實質的東西而感到開心。

### 統計方法
主要聚焦於視覺化圖表以及解釋圖表的範疇，並沒有使用到過多的模型。

### 研究探討
主要聚焦於找尋各種可以解釋圖表以及關聯性的資料，成功探討了各種音樂間的關聯性。
