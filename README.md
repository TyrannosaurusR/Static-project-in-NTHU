# Static-project-in-NTHU
此份 repo 將紀錄我在清大做過的各種數據分析專案

## 如何在特定條件透過調整習慣
此部分為我修習統研所的統計學習後所作出的專案作品，此份專案作品使用了 Kaggle 上記錄了學生的每天時間分配以及其學習表現的資料，較為可惜的是這是一份虛擬的資料

### 資料處理
由於這是一份虛擬的資料，所有的變數都只呈現很完美的常態分佈，且相關性都只與學習表現有關。

### 研究過程與結果
我們利用嶺回歸來探討學習的邊境遞減效應，以及利用多項式回歸得出最好的模型後，再利用線性規劃得出理論上的時間分配最佳值

### 心得
這份作業合作起來沒有那麼開心，組員一開始沒有很積極。當時一開始，我是想要使用真實的資料的，於是我翻過很多資料集以及論壇，可惜最後時間真的來不及，我才也接受了使用這份虛擬資料。
